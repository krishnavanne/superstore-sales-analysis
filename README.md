# superstore-sales-analysis
Superstore Sales Analysis using Python to identify sales trends, profit patterns, and business insights.

## 📌 Project Overview
This project analyzes a retail Superstore dataset to understand sales performance, profit trends, and business insights using Python.

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 🔍 Key Steps
- Data Cleaning and Preprocessing
- Feature Engineering (Year, Month, Shipping Days)
- Exploratory Data Analysis (EDA)
- Data Visualization

## 📊 Key Insights
- Technology category generated the highest sales
- Some products resulted in losses (negative profit)
- Higher discounts reduce overall profitability
- Sales and profit vary across different regions

## 📌 Conclusion
This analysis helps businesses improve decision-making by identifying profitable areas and reducing losses.
